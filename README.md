## 💼 Sobre o Projeto

> 💡 **Design Editorial Digital, Bento Layout e reaproveitamento de codigo**

esse projeto do almanaque fontoura foi desenvolvido na grade de ADS. a ideia principal foi pegar o conteudo de um almanaque antigo de farmacia (edição de 1964) e transformar numa revista digital com cara de the new york times ou vogue. o maior desafio foi reutilizar obrigatoriamente as classes css da minha AV1 e mudar todo o visual sem quebrar o html.

🔧 **Destaques Tecnicos:**
- **Bento Grid:** organização dos cards num layout assimetrico super moderno.
- **Light/Dark Mode:** botão que muda as variaveis `:root` do css pra alternar entre o modo noturno e o modo claro na hora.
- **Intersection Observer:** o javascript fica vigiando a tela e faz os cards aparecerem com um efeito de fade in quando a gente rola a pagina.
- **Reuso de Codigo:** mantive os prefixos obrigatorios (`sc_rafael_`) da atividade anterior e reescrevi so a casca visual.
